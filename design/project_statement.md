# TicketFlow - Declaración del Proyecto

## Tema
**TicketFlow** es una aplicación web diseñada para facilitar la compra de entradas para diversos eventos como conciertos, obras de teatro y películas. La plataforma permite a los usuarios explorar eventos disponibles, ver información detallada y comprar entradas de forma segura. Los administradores pueden gestionar el catálogo de eventos.

## Requisitos Funcionales
1.  **Autenticación de Usuario**: Los usuarios pueden registrarse e iniciar sesión.
2.  **Catálogo de Eventos**: Los usuarios pueden ver una lista de eventos con filtros (Categoría, Fecha).
3.  **Detalles del Evento**: Los usuarios pueden ver todos los detalles de un evento (Descripción, Ubicación, Precio).
4.  **Compra de Entradas**: Los usuarios pueden comprar entradas para un evento.
5.  **Historial de Compras**: Los usuarios pueden ver sus reservas pasadas.
6.  **Gestión de Eventos (Admin)**: Los administradores pueden crear, actualizar y eliminar eventos.
